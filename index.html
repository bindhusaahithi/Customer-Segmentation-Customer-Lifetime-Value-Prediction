Data Science Project

Customer Segmentation & Spend Analysis

An advanced customer analytics project using RFM analysis, K-Means clustering, spend estimation, and business-focused visual storytelling on e-commerce data.

Explore Project View GitHub Repo

Project Overview

This project analyzes customer purchasing behavior using the Online Retail dataset. The workflow includes data cleaning, RFM feature engineering, outlier handling, K-Means clustering, and a machine learning model to estimate customer spending behavior.

Dataset

Online Retail (UCI)

Tech Stack

Python, Pandas, Scikit-learn

Core Method

RFM + K-Means

Model

Random Forest

Workflow

1

Data Cleaning

Removed cancelled transactions, missing customer IDs, and invalid quantity/price records.

2

Feature Engineering

Created total price and RFM features: Recency, Frequency, and Monetary.

3

Segmentation

Used K-Means clustering to group customers by purchasing behavior.

4

Spend Modeling

Built a Random Forest model to estimate customer spending patterns.

Key Insights

High-Value Segment

A relatively small customer group contributes a major share of total revenue.

Frequency Matters Most

Purchase frequency is more important than recency in estimating spend behavior.

Business Value

The analysis can support targeted marketing, retention strategy, and segmentation decisions.
